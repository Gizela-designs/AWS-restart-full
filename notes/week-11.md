# Amazon Q Developer
Overview Amazon Q Developer is a generative AI-powered conversational assistant built on Amazon Bedrock that helps with software development across the entire lifecycle.

# Key Features
- Agentic Coding

- Breaks down prompts into logical implementation steps

- Autonomously generates code, tests, and API integrations

- Implements features end-to-end with minimal human intervention

# Code Generation & Assistance
- Real-time code suggestions in 25+ programming languages
- Supports Java, Python, JavaScript, and more
- Context-aware completions from snippets to full functions

# Recent Capabilities (December 2024)
